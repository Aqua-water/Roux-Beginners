# Roux-Beginners

[教程网站](https://aqua-water.github.io/Roux-Beginners/%E7%AE%80%E4%BB%8B.html)

## 介绍

这个项目实现了一个在线的魔方桥式解法入门教程，其特点是不需要任何的魔方基础，而且可以观察虚拟魔方的转动来理解公式。教程提供了大量的细节，以使之适合自学。

## 参与贡献

如果你想成为开发者，可以在本地完成部署与编辑，然后提交请求。了解一些git操作的话，将会有所帮助。

简要说明如下：

下载并部署[mdbook](https://github.com/rust-lang/mdBook/releases/)程序，克隆本仓库到本地。

完成后，在Roux-Beginners文件夹下打开终端，运行:

```
mdbook serve --open
```

这样，你可以在对Markdown文件进行编辑的同时，查看网站渲染的内容的变化。这对正确编写虚拟魔方的代码非常有帮助。修改后即可提交request。

如果你有任何的疑问或建议，也欢迎在本项目下提交issue。
